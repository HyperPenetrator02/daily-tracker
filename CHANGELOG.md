# Changelog

All notable changes to StatMaxer RPG OS will be documented in this file.

## [1.1.0] - 2026-01-20

### Added
- Native Android notifications with LocalNotifications plugin
- Mobile-optimized UI for tall screens (2412x1080)
- Comprehensive responsive design for all screen sizes
- Hamburger menu for mobile navigation
- Touch-optimized controls (44-48px minimum targets)
- Background alarm support (works when app is closed)
- Extensive debugging and error handling

### Changed
- Improved notification reliability on Android
- Enhanced mobile layout with reduced spacing
- Optimized font sizes for better readability on small screens
- Updated build process to include all CSS files

### Fixed
- Notification permissions on Android
- Sidebar visibility on mobile devices
- APK build process for automated releases
- Service worker caching for offline support

## [1.0.0] - 2026-01-18

### Added
- Initial release of StatMaxer RPG OS
- Three core views: Quest Log, Stat Matrix, Character
- XP system with level progression
- Streak multipliers (1.5x for 3+ days)
- Six RPG categories (Strength, Intelligence, Agility, Discipline, Vitality, Charisma)
- Hardcore alarm system with snooze penalties
- Radar charts for category visualization
- Progressive Web App support
- Offline functionality with Service Worker
- 31-day habit tracking matrix
- Dark cyberpunk theme
- Privacy-focused (100% offline, no tracking)

### Features
- Custom habit creation with icons
- Alarm scheduling per habit
- Hardcore mode for critical habits
- XP penalty system for snoozing
- Character progression and stats
- Mobile-responsive design
- GitHub Pages deployment
- Automated APK builds via GitHub Actions
